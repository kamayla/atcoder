# 問題文

正整数N が与えられるので、N がハッピー数であるかを判定してください。

ハッピー数とは、以下の操作を有限回繰り返すことで
1 になる非負整数を指します。

自身を、十進法表記の各桁の数字の二乗和を取った整数に置き換える。

## 制約

N は1 以上2026 以下の整数

## 考え方

有限回というのがどの程度なのかを算出するのがポイントで、それがわかれば愚直に問題文通りの操作を有限回繰り返せば良い。

最大値が2026なので最大四桁の各桁の自乗和の合計の最大値は、9^2 \* 4 = 324である。

さらに、この三桁についても同様の操作を行ったときの最大値は三桁なので、9^2 \* 3 = 243である。

さらにこの三桁についても同様の、、、と繰り返すわけなのでこれ以降の算出結果243を超えることはない。

つまり、243回以内で1になるか、243以下の値のどこかでループすることになるので、243回繰り返しても1にならなければループしていると判断してハッピー数ではないと判定すれば良い。(243以下の整数は243個しかない。)

よって、厳密な証明は省略するが、300回程度繰り返せば十分に判定できることになります。
