# メモ

## 問題文

N 個の座席が横一列に並んでいます。各座席には最大で1 人まで座ることができます。以下の条件を満たすようにM 人の人を座席に座らせることができるかどうか判定してください。

> 隣り合う 2 つの席の両方に人が座ってはいけない。

## この問題のポイント

どういう状況が禁じられているのかを正確に理解することが重要。

以下のように隣り合う両方の席に人が座っている状況は禁じられている。

> ◯◯

そして、以下のように隣り合う両方の席のどちらかに人が座っている状況は許されている。

> ◯× or ×◯

## 解法

この問題は、◯×を一塊と捉えてこの塊がN個の席の中にいくつ入るかを考えると簡単に解ける。

例えば、M=3のとき、◯×を一塊と捉えると、以下のように考えることができる。

> ◯×◯×◯×
> このように、M人の人を座らせるためには、M個の◯×の塊が必要であることがわかる。

したがって、M人の人を座らせるためには、M個の◯×の塊が必要であることがわかる。

よって、m <= ceil(N / 2) であれば、M人の人を座らせることができることがわかる。

ceilで繰り上げている理由は、Nが奇数のときに、最後の席に人が座ることができるからである。

例えば、N=5のとき、以下のように考えることができる。

> ◯×◯×◯
